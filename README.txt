Este es un pequeño documento orientativo para poder ejecutar el minijuego con exito y una breve explicación de algunas opciones del juego.
Es necesario tener un IDE (como el clásico Visual Studio Code) para poder ejecutarlo.

Ya está creado el archivo .json donde directamente se abrirá el Google Chrome donde directamente se ejecutará el juego.
Para una mejor experiencia siempre recomiendo utilizar la función de "Pantalla Completa" ya que el juego está pensado para jugarlo en los ordenadores del Politecnics que tienen una dimensiones de 1920x1080.

Hay botones del juego que no funcionan A PROPOSITO debido a que si revisais el código vereis que algunos enlaces llevarían directamente a otro HTML; ese es el motivo por el cual el link del botón está puesto con un hastag (#) y no con el link pertinente y por eso está la linea comentada.
